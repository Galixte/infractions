Recalculate Rule Penalties

You may make changes to Rules and wish to reapply the current Rules (i.e. any Rules currently in effect will be reverted and then all new Rule penalties will be applied to users who have enough non-expired points).

Warning

This process cannot be undone! All currently applied Rule penalties will be reverted (and users will be removed from the Target Groups of all Rule Penalties applied to them). Then, all Rule penalties will be recalculated (and users will be placed in groups based on the current Rules set).
